git的使用

概念

git：版本控制工具

git的三个区

  • 工作区：红色
  • 暂存区： 绿色
  • 版本区（库）master
    git命令
    git添加
  • git status 当前状态
    • 加号：增加了几个
    • ~：修改了几个
    • 减号：删除了几个
  • git add 添加到暂存区
    • git add . 全部添加到暂存区
  • git commit 添加到版本区
    • git commit m “add demo1.html” 添加注释
  • git commit -a -m “” 直接添加到版本区和注释
  • git log 查看历史
    git对比
  • git diff 工作区与暂存区的对比
  • git diff –cached 暂存区与版本区的对比
  • git diff –staged 跟上面相同暂存区与版本区的对比
    git撤销

github：社交平台、远程仓库、开源项目
